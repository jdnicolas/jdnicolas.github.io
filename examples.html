Example data can be obtained from the supplementary material of the following article:

J.-D. Nicolas, M. Bernhardt, A. Markus, F. Alves, M. Burghammer and T. Salditt, Scanning X-ray Diffraction on Cardiac Tissue: Automatized Data Analysis and Processing, submitted (2017).

You are free to analyze the data in any way. An analysis example using Principal Component Analysis is built into the Matlab toolbox.
